# BG Hub Backend

NestJS API for BG Hub.

## API Docs

- **OpenAPI spec (JSON):** [http://localhost:3000/openapi.json](http://localhost:3000/openapi.json)
- **Scalar UI:** [http://localhost:3000/docs](http://localhost:3000/docs)

## Local dev

```bash
npm run start:dev
```

Set `DATABASE_*` and other options in `.env` (see `.env.example`).
